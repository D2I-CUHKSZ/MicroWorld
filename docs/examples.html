Signal Summary

What this experiment shows

This case is useful because the result is easy to explain: people react very differently when a scientific breakthrough is still exciting, when replication becomes contested, and when a more evidence-based consensus starts to form.

Influence Reading

Directional influence in plain language

The interesting public-facing point is not a raw score. It is that influence is often one-way: some actors set the framing, while others mostly react, reinterpret, or amplify. LightWorld tries to make that asymmetry readable.
